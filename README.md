# Superstore Sales Dashboard

##Project Overview

This project analyzes a Superstore dataset to understand sales performance, profit trends, and product insights.

##Tools Used

* Microsoft Excel (Pivot Tables)
* Google Looker Studio (Dashboard)

## Files Included

* Excel file with pivot table analysis
* Dashboard screenshot
* Pivot table screenshots

## Key Insights

* West region has the highest sales contribution
* Technology category generates maximum revenue
* Phones and Chairs are top-performing sub-categories

##Purpose

This project is created to practice data analytics and dashboard building skills.
